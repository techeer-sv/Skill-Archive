# Known Company DB

> 이 정보는 공식 채용 페이지, 기술 블로그, 면접 후기를 종합한 참고 자료입니다.
> **실제 프로세스와 다를 수 있으며**, 사용자 확인을 거쳐 사용합니다.

## 한국 빅테크

### 카카오 (Kakao)
- **기준일:** 2025-03
- **출처:** 카카오 공식 채용 페이지, 프로그래머스 공채 정보
- **프로세스:**
  1. 서류 심사 — resume-review:default
  2. 1차 코딩 테스트 (5문제/5시간) — coding-test:algorithm
  3. 2차 코딩 테스트 (감독관) — coding-test:algorithm

  5. 임원 면접 — interview:behavioral
- **특성:**
  - 평가 중점: 알고리즘
  - 면접 스타일: 구조화
  - 고유 가치: 일반
  - 코딩 테스트 플랫폼: 프로그래머스
  - 특이사항: 블라인드 채용, 2차 코테는 감독관 하에 진행
- **주의:** 공채와 수시 채용 프로세스가 다를 수 있음

## 스타트업 일반 프로필

회사명이 Known DB에 없고 스타트업으로 판단되는 경우, 규모에 따라 적용:

### 초기 스타트업 (50명 미만)
- **프로세스:**
  1. 서류 심사 — resume-review:default
  2. 과제/포트폴리오 — coding-test:take-home 또는 portfolio-review:default
  3. 대표/CTO 면접 — interview:technical + interview:behavioral
- **특성:** 2-3라운드, 빠른 결정, 미션 정렬 중시

### 성장기 스타트업 (50-200명)
- **프로세스:**
  1. 서류 심사 — resume-review:default
  2. 코테/과제 — coding-test:algorithm 또는 coding-test:take-home
  3. 기술 면접 — interview:technical
  4. 컬처핏 면접 — interview:behavioral
- **특성:** 3-4라운드, 실무 중심

### 스케일업 (200-500명)
- **프로세스:**
  1. 서류 심사 — resume-review:default
  2. 코딩 테스트 — coding-test:algorithm
  3. 기술 면접 — interview:technical
  4. 행동 면접 — interview:behavioral
  5. 임원 면접 — interview:behavioral
- **특성:** 4-5라운드, 빅테크에 가까운 구조
